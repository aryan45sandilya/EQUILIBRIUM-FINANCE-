## ✨ Features

| | Feature | Details |
|---|---|---|
| 🏠 | **Group Management** | Create groups, invite by email, manage roles |
| 💸 | **Flexible Splits** | Equal · Percentage · Exact · Shares |
| 🧠 | **Debt Simplification** | Greedy Min-Cash-Flow — minimum transactions |
| 🔔 | **Real-time Notifications** | Socket.IO — instant alerts |
| ✅ | **One-click Settlement** | Mark paid, balances update instantly |
| 📊 | **Spend Analytics** | Category and member breakdown |
| 🔐 | **Clerk Auth** | Passwordless, OAuth, secure JWT |

## 🚀 Local Setup

```bash
git clone https://github.com/aryan45sandilya/EQUILIBRIUM-FINANCE-.git
cd EQUILIBRIUM-FINANCE-/src

# Backend
cd backend && cp .env.example .env && npm install && npm run dev

# Frontend (new terminal)
cd frontend && cp .env.example .env.local && npm install && npm run dev
